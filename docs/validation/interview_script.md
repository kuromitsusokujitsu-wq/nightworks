# 行政書士 課金意思ヒアリング 台本

目的：プロダクトを深掘りする前に「行政書士が月額で払うか」を実地データで確認する。M0（最優先・並行）マイルストーン。撤退ライン＝8週で課金意思0人。

## 進め方

- 対象：補助金を扱う行政書士 3-5人。CEO のコネ起点＋行政書士会公開名簿からの少数コールド。
- 形式：15-20分。売り込みではなく課題ヒアリングの姿勢。
- 記録：各回 `interview_log.md` に1行追記。

## 核質問（必ず全て聞く）

1. 補助金1件の申請書作成に、平均どれくらいの工数（時間）がかかりますか？
2. 1件あたりの報酬はおおよそいくらですか？
3. 申請書作成の工程で「一番苦痛／時間が溶ける」のはどこですか？（要件読み込み／事業計画の文章化／加点対策／差し戻し対応 等）
4. その工程の下書きをAIが7割完成させて出してくれるツールがあったら、月いくらまでなら払いますか？
5. 今、補助金業務で使っているツール／不満は何かありますか？

## 観察ポイント

- 「払う」と言った行政書士の具体額と前提条件（事務所規模・年間件数）。
- 痛みの工程が drafter/researcher で本当に解けるか。
- 価格感が経営基盤の前提（¥3-5万）と整合するか。

## ゲート

課金意思を明示する行政書士が出るまで、プロダクトは最小実装に留める（過剰投資の防止）。
