# 株式会社X-Style 運用リポジトリ

X-Style の経営基盤とエージェント組織を定義する運用リポジトリ。コードベースではなく、1人法人を Claude Code 駆動で回すための「会社の憲法と組織」を格納する。

## 事業

行政書士向け **補助金申請書AI下書き支援ツール**（月額サブスク）。

- 顧客は **行政書士のみ**。提供物は行政書士が使う下書き支援ツール。
- 事業者へ直接、報酬を得て申請書を作成することは**絶対禁止**（行政書士法1条の2・19条）。
